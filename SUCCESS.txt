🧪 TEST THE API / API ٹیسٹ کریں:

Method 1 / طریقہ ١ - Swagger UI (Easiest!):
   
   1. Open: http://localhost:8000/api/docs/
   2. Click: POST /api/auth/register/
   3. Click: "Try it out"
   4. Enter data:
      {
        "email": "test@example.com",
        "username": "testuser",
        "full_name": "Test User",
        "password": "Test123!@#",
        "password_confirm": "Test123!@#"
      }
   5. Click: "Execute"
   6. See response! ✅

Method 2 / طریقہ ٢ - Admin Panel:
   
   1. Open: http://localhost:8000/admin/
   2. Login with superuser credentials
   3. Manage all data from here!

📚 AVAILABLE ENDPOINTS / دستیاب APIs:

Authentication / تصدیق:
  ✓ POST /api/auth/register/       - Sign up
  ✓ POST /api/auth/login/          - Sign in
  ✓ POST /api/auth/logout/         - Sign out
  ✓ POST /api/auth/password-reset/ - Reset password

Users / صارفین:
  ✓ GET  /api/users/me/            - My profile
  ✓ GET  /api/users/               - All users (Admin)

Products / مصنوعات:
  ✓ GET  /api/products/            - List products
  ✓ POST /api/products/            - Create product (Manager)

Orders / آرڈرز:
  ✓ GET  /api/orders/              - My orders
  ✓ POST /api/orders/              - Create order

And many more... / اور بہت کچھ...
