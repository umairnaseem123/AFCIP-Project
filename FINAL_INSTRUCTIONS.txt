❌ IF YOU SEE ERRORS / اگر آپ کو errors نظر آئیں
═══════════════════════════════════════════════════════════════════════

   ERROR: "No module named 'django'"
   
   ❌ Cause: Virtual environment not activated
      وجہ: Virtual environment ایکٹیو نہیں
      
   ✅ Fix: Use RUN_SERVER.bat (it handles this automatically)
      حل: RUN_SERVER.bat استعمال کریں (خودکار حل)
      
   ---
   
   ERROR: "No module named 'whitenoise'"
   
   ❌ Cause: Package not installed in venv
      وجہ: Package venv میں نہیں
      
   ✅ Fix: RUN_SERVER.bat will install it automatically
      حل: RUN_SERVER.bat خودکار install کر دے گا
      
   OR manually:
   یا دستی:
   
      .\venv\Scripts\Activate.ps1
      pip install whitenoise
      
   ---
   
   ERROR: "Port is already in use"
   
   ❌ Cause: Port 8000 already being used
      وجہ: Port ٨٠٠٠ پہلے سے استعمال میں
      
   ✅ Fix: Use different port
      حل: مختلف port استعمال کریں
      
      python manage.py runserver 8001
      
      Then open: http://localhost:8001/api/docs/
      پھر کھولیں

📱 AVAILABLE API ENDPOINTS / دستیاب API Endpoints
═══════════════════════════════════════════════════════════════════════

   Authentication / تصدیق:
   • POST /api/auth/register/          - Register new user
   • POST /api/auth/login/             - Login
   • POST /api/auth/logout/            - Logout
   • POST /api/auth/password-reset/    - Request password reset
   
   Users / صارفین:
   • GET  /api/users/me/               - My profile
   • GET  /api/users/                  - List all users (Admin)
   
   Products / مصنوعات:
   • GET  /api/products/               - List products
   • POST /api/products/               - Create product (Manager)
   • GET  /api/products/{slug}/        - Product details
   
   Orders / آرڈر:
   • GET  /api/orders/                 - My orders
   • POST /api/orders/                 - Create order
   • GET  /api/orders/{id}/            - Order details
   
   Payments / ادائیگی:
   • GET  /api/payments/               - My payments
   • GET  /api/payments/{id}/          - Payment details
   
   Transactions / لین دین:
   • GET  /api/transactions/           - My transactions
   • GET  /api/transactions/{id}/      - Transaction details
   
   Notifications / اطلاعات:
   • GET  /api/notifications/          - My notifications
   • POST /api/notifications/{id}/mark_read/ - Mark as read
