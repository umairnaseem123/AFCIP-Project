📝 MANUAL START:

  1. Activate virtual environment:
     > venv\Scripts\activate

  2. Install dependencies:
     > pip install -r requirements/dev.txt

  3. Setup database:
     > python manage.py migrate

  4. Create admin user:
     > python manage.py createsuperuser

  5. Start server:
     > python manage.py runserver
